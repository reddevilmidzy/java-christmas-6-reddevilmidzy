# 구현 기능 목록


## 기능

* [ ] 총주문 금액 10,000 미만인 경우 이벤트 적용하지 않음
* [ ] 크리스마스 디데이 할인
  * 기간: 2023.12.1 ~ 2023.12.25
  * 정책: 1000원에서 시작하여 날이 지날수록 100원 증가.
  * 총주분 금액에서 할인금액 제외
* [ ] 평일 할인
  * 기간: 2023.12.1 ~ 2023.12.31
  * 정책: 일~목에 디저트메뉴 1개당 2023원 할인
* [ ] 주말 할인
  * 기간: 2023.12.1 ~ 2023.12.31
  * 정책: 금,토에 메인 메뉴 1개당 2023원 할인
* [ ] 증정 이벤트
  * 기간: 2023.12.1 ~ 2023.12.31
  * 정책: 할인 전 총 주문 금액이 12만원 이상인 경우 샴페인 1개 증정

* [ ] 베지 부여
  * 정책: 총혜택 금액을 확인하여 배지 증정
    * 5천 원 이상: 별
    * 1만 원 이상: 트리
    * 2만 원 이상: 산타



## 입력

* [x] 방문 날짜 입력
* [x] 주문할 메뉴와 개수 입력
  * 입력폼: [메뉴1]-[개수1],[메뉴2]-[개수2]
  * ex) 타파스-1,제로콜라-1


## 출력

* [ ] 주문 메뉴
* [ ] 할인 전 총주문 금액
* [ ] 증정 메뉴
  * 없는 경우 '없음'
* [ ] 혜택 내역
  * 없는 경우 '없음'
* [ ] 총혜택 금액
* [ ] 할인 후 예상 결제 금액
* [ ] 12월 이벤트 배지


## 예외

* [ ] 유효하지 않은 날짜인 경우
* [ ] 등록되지 않은 메뉴인 경우
* [ ] 메뉴 개수가 유효하지 않은 숫자인 경우
* [ ] 총 주문 메뉴 수가 20을 초과한 경우
* [ ] 음료만 주문한 경우
